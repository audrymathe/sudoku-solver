# sudoku-solver

This takes any size of sudoku input, and returns the solution.

This is a c++ code , the input must be of the form


0 4 0 0 0 0 1 7 9
0 0 2 0 0 8 0 5 4
0 0 6 0 0 5 0 0 8
0 8 0 0 7 0 9 1 0
0 5 0 0 9 0 0 3 0
0 1 9 0 6 0 0 4 0
3 0 0 4 0 0 7 0 0 
5 7 0 1 0 0 2 0 0 
9 2 8 0 0 0 0 6 0 

for a 9 x9



then for a 16 x 16 numbers must range from{0,1.....-9,A,......F}
and for a 25 x25 {0,1,.......,9,A,.......P }
